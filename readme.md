## Backend install ##
1. composer install
2. php artisan migrate:refresh --seed
3. php artisan serve
4. set QUEUE_DRIVER=Beanstalkd in env
5. install drivers for php-ffmpeg

### Backend requirements ###
1. PHP >= 5.6.4
2. MYSQL 5.7
